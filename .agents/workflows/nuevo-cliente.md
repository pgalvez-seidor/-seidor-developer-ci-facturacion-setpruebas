# Workflow: Nuevo Cliente

## Pasos

1. Lee el archivo `CONTRIBUTING.md` completo para entender el flujo de integración.

2. Sigue la **Fase 1** de CONTRIBUTING.md: haz las preguntas guiadas al usuario, una por una, en el orden indicado.

3. Una vez recopilada toda la información, sigue la **Fase 2** de CONTRIBUTING.md: genera todos los archivos automáticamente.

4. Sigue la **Fase 3** de CONTRIBUTING.md: ejecuta la validación post-setup (test API + test portal).

5. Confirma al usuario que el cliente está listo.
